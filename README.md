Remote Watering System

A smart remote watering system that allows users to control plant watering anytime and anywhere through the internet.
This project helps reduce manual work, save time, and lower labor costs with real-time remote control.

+ Features
- Remote watering control
- Real-time communication using Socket.IO
- Automatic watering support
- Simple web interface
- Lightweight Flask backend
+ Technologies Used
- Python
- Flask
- Socket.IO
- HTML
- CSS
